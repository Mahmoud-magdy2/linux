حل مشاكل تثبيت البرامج على لينكس

ما هو Flatpak؟

Flatpak هو نظام حزم مستقل يسمح لك بتثبيت التطبيقات على مختلف توزيعات لينكس بدون مشاكل توافق.

1. تثبيت أو تحديث Flatpak

sudo apt update && sudo apt install flatpak -y

2. إضافة Flathub

            flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

3. تثبيت البرامج باستخدام Flatpak

لتثبيت Discord:

flatpak install flathub com.discordapp.Discord

لتثبيت Zoom:

flatpak install flathub us.zoom.Zoom
